# Day-14-Higher-Lower-project
Who has more followers? Nicki or Cardi? 

p.s I've graduated from beginners, yay! Let's venture into intermediate territory

The objective of the game is to guess which of the given personalities has more instagram followers. A or B

The number of correct guesses are recorded and displayed in every round

If guessed correctly, the chosen personality is then taken to the next round and pitted against a new personality.

A wrong guess will end the game and display your final score.
